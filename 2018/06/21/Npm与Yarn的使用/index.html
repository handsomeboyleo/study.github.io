NPM&YARN

npm常用的命令

全局安装：默认安装在Node.js环境的node_modules目录下，全局安装的包一般用于命令行工具

本地安装：一般用于开发某种具体的功能

1、安装包（若不指定版本号 默认安装最新版本）

npm install -g 安装包名称     （全局安装）

npm install 安装包名称      （本地安装）
例： npm install es-checker     //安装es6
例： npm install

2、卸载包

npm uninstall -g 安装包名称     （全局卸载）

npm uninstall 安装包名称      （本地卸载）
例： npm uninstall es-checker     //卸载es6

3、更新包 （更新到最新版本号）

npm updata -g 安装包名称

4、添加依赖

--save   （向生产环境添加依赖） 生产环境是项目部署上线之后的服务器环境
--save-dev   （向开发环境添加依赖） 开发环境是平时开发使用的环境
//例子：npm install es-checker --save
//例子：npm install es-checker --save-dev

安装服务器自动重启

npm install -g nodemon

npm安装包的步骤

1、初始化

npm init //初始化 生成package.json文件

2、安装包

npm install 安装包名称      （本地安装）

3、若存在package.json文件 和 package-lock.json文件，恢复安装

npm install

Yarn

安装yarn工具

npm install -g yarn

yarn中的命令

1、初始化包

npm init  //npm中的初始化方式
yarn init  //yarn中初始化方式

2、安装包

npm install xxx --save //npm中的安装方式
yarn add xxx  //yarn中的安装方式

3、移除包

npm uninstall xxx  //npm中的移除方式
yarn remove xxx  //yarn中的移除方式

4、更新包

npm update xxx
yarn upgrade xxx

5、安装开发依赖的包

npm install xxx --save-dev
yarn  add xxx --dev

6、全局安装

npm install -g xxx
yarn globar add xxx

7、设置下载镜像的地址

npm config set registry url
yarn config set registry url

8、安装所有依赖

npm install
yarn install

9、执行包

npm run
yarn run

切换源

安装nrm

npm install nrm

查看源

nrm ls

切换

nrm xxx
